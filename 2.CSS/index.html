Level 1

CSS - Cascading Style Sheets

  • CSS styles the look & feel of webpages.
  • Rule structure → selector { property: value; }
  • Default stylesheet filename → style.css

Basic Syntax

  • Selector → chooses element(s) (e.g., h1)
  • Property → aspect to change (e.g., color)
  • Value → setting for property (e.g., red)

Including CSS

  • Inline CSS
    • <h1 style="color:red;">Red Heading</h1>

  • Internal CSS
    • <style>
  h1 { color: red; }
</style>

  • External CSS
    • <link rel="stylesheet" href="style.css">


Selectors

  • Universal Selector
    • * { margin:0; padding:0; }

  • Type Selector
    • h1 { color:blue; }

  • Class Selector
    • .myClass { font-size:20px; }

  • ID Selector
    • #myId { background:yellow; }


Color

  • Text color
    • color: red;
    color: blue;

  • Background color
    • background-color: green;
    background-color: yellow;

  • Color Systems
    1. Named colors → red, blue, green, yellow

    2. RGB
      3. color: rgb(255,0,0);
      color: rgb(0,255,0);

    3. Hexadecimal
      4. color: #FF0000;
      color: #00FF00;


Text Properties

  • Font family
    • font-family: Arial, sans-serif;

  • Font size
    • font-size: 16px;
    font-size: 1.5em;

  • Line height
    • line-height: normal;
    line-height: 20px;

  • Font weight
    • font-weight: bold;
    font-weight: 400;
    font-weight: 100-900;

  • Text align
    • text-align: left;
    text-align: center;

  • Text decoration
    • text-decoration: underline;
    text-decoration: line-through;

  • Text transform
    • text-transform: uppercase;
    text-transform: lowercase;
    text-transform: capitalize;

Level 3

Units in CSS

  • Absolute units → px, in, cm
  • Relative units → em, rem, %
  • Viewport units → vw, vh


Positioning

  • Static
    • position: static;

  • Relative
    • position: relative; top: 10px; left: 20px;

  • Absolute
    • position: absolute; top: 0; left: 0;

  • Fixed
    • position: fixed; bottom: 0; right: 0;

  • Sticky
    • position: sticky; top: 0;


Background Image

  • Set → background-image: url('image.jpg');
  • Size → cover, contain, auto
  • Position → center, top right
