Level 1: HTML Basics

HTML

  • HyperText Markup Language
  • HTML is the code that is used to structure a webpage and its content.
  • The component used to design the structure of a webpage is called HTML Tags.
  • HTML tag is parent of head & body tags.
  • Most HTML tags have an opening tag and a closing tag in between.
  • Some tags have content in between, eg- <br>.
  • We can use inspect element/view source to see/edit the HTML of any webpage.

HTML is NOT case sensitive

  • <html> = <HTML>
  • <p> = <P>
  • <head> = <HEAD>
  • <body> = <BODY>

Comments in HTML

Comments are not displayed in the browser but are useful for developers.

This is part of code that should not be parsed.

  <!-- This is a comment -->

Boilerplate

  <!DOCTYPE html>
  <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Document</title>
    </head>
    <body>
      <!-- Content goes here -->
    </body>
  </html>
